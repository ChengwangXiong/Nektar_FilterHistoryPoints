LocalRegions

LocalRegion is-a StdRegion has-a SpatialDomain

This library requires SpatialDomains, StdRegions, and LibUtilities. It presents expansions that live over elements in world space. The element (StdRegion) also includes the geometry (SpatialDomain), so an example of use are methods that integrate and take the derivative.
