LibUtilities

This library is independent of all others (except for Boost, but that is considered standard). It contains general C++ building blocks that support the other libraries. This library contains:

  • Linear algebra routines
  • Block-matrix routines
  • Data managers and memory pools
  • Polynomial manipulation routine
